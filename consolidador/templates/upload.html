Fluxo operacional

O que acontece depois do envio

  1. O lote recebe identificacao unica com remetente, data e horario.
  2. O sistema avalia Saude e Odonto separadamente com base nas planilhas de referencia.
  3. O admin gera os consolidados e acompanha inconsistencias sem misturar os dois fluxos.

Boas praticas

Como evitar retrabalho

  • Use o campo correto para cada tipo de remessa.
  • Evite arquivos-modelo sem dados reais no mesmo lote.
  • Padronize nome e e-mail para facilitar localizacao posterior.

Leitura rapida

Quando usar observacao

  • Quando a remessa complementar um protocolo anterior.
  • Quando houver excecoes que o time administrativo precise conferir.
  • Quando for necessario contextualizar ajustes de titulares ou dependentes.
